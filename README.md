Google Login Module
===================


Google Firebase를 이용한 React Native Login
-----------------------------------------


구글 아이디를 통해 앱 내 로그인 구현

#### $ yarn add or npm install -save @react-native-community/google-signin
#### $ cd ios 
#### $ pod install 
#### $ cd ..

##### 작업 시 어려움은 크게 없었으나 Firebase에서 프로젝트 생성 시 WebClient ID는 저장하는 것이 편하다.
